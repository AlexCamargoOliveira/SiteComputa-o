DESENVOLVIMENTO WEB
Muitos não conseguem imaginar uma vida sem internet, sem pesquisas, redes sociais e estar antenado aos acontecimentos que ocorrem ao redor do mundo. Uma entre os principais influenciadores de todo desenvolvimento tecnológico atual para a área da educação, saúde, conhecimento e diversas outras. Para quem desconhece sua história é até difícil imaginar que o verdadeiro motivo por trás de sua origem está muito além da internet que temos hoje.
HISTÓRIA DA INTERNET



O inicio da história de seu surgimento ocorreu nos Estados Unidos, em meados da década de 60, onde o poderia militar durante a Guerra Fria era fator decisivo para vencer a mesmo. Em prol de evitar possíveis perdas de seus documentos em razão dos ataques, cientistas dos EUA criaram um sistema de redes telefônicas que interligavam seus centros de comando e de pesquisa bélica. Onde as informações eram divididas em pequenos pacotes de dados que permitiam a remontagem da informação original, codificando assim suas informações, para caso houvesse a perda de uma de suas unidades.



Nos anos 70, esta tecnologia foi disponibilizada para uso das comunidades acadêmica mundial, criados por eles então os famosos e-mails, através do desenvolvimento de protocolos, um experimento entre agências como a DARPA, entre eles, o TCP/IP e o FTP (File Transfer Protocol, protocolo para transferências de arquivos para a web entre um servidor e um cliente), com a utilização do termo “Internet” em dezembro de 1974. Após a ARPANET(nome ao qual foi dado ao experimento), começou a ganhar fundos e ser desenvolvida, nos anos 90, para fins comerciais, quando começou a popularização desta rede com o advento do modelo de “hipertexto” em 1989.



Em 1991, foi criado o Hypertext Markup Language(HTML), por Tim Burns Lee, ao qual poderia ser utilizado para criar novos documentos de texto que poderiam ser compartilhados pela rede da internet. Em 1992, Tim, desenvolveu o chamado WWW (World Wide Web). Antes, quando apenas era possível circular textos pela internet, com o WWW se tornou possível a transmissão de imagens, sons e vídeos pela rede, possibilitando maior procura pelo público em geral, visto o grande investimento para este tipo de uso, com a vindo de navegadores que pudessem interpretar com mais desenvoltura estes códigos, como o Netscape(1994). Foi então que, em 1994, foi criado o Cascading Style Sheets(CSS), por Hakon Wium Lie e Bert Bos, com o intuito de facilitar a programação de sites, que na época era muito complexa; ganhando destaque entre os programadores em 1997, como a linguagem que determina a aparência de uma página para a web.



Estas duas linguagens, HTML e CSS, serão o foco de nosso estudo para o desenvolvimento de sites para a web, iniciando com a introdução a HTML.
HTML
Primeiramente criamos um documento HTML através de um software multiplataforma de edição de texto, como por exemplo o Notepad++ e o Sublime Text, criando um novo documento de tipo .html.

Para começar entenderemos o que é uma tag. Tags são estruturas de linguagem de marcação contendo instruções, tendo uma marca de início (“<”, sinal de menor) e outra de fim (“>”, sinal de maior). Como por exemplo a tag com a instrução html, que possui um início e um fim, onde dentro destas marcações haverá o conteúdo feito através da linguagem html que será lido pelo navegador web. Alguns tags não apresentam a necessidade de declarar seu fim, como veremos a frente.

Para reconhecimento do tipo de documento a ser lido por um navegador da web, digita-se a tag: !DOCTYPE html.

Após a tag html de início e fim e o mesmo para as tags head (onde serão digitados metainformações que poderão ser visualizadas por outros sites) e body (simplesmente o “corpo” do site, onde haverá todo o conteúdo que será lido e visivelmente mostrado pelo navegador a quem acessar esta página).

Iremos preencher as informações necessárias nos campos vazios de ambas as tags.
HEAD

Na tag head poderão ser adicionados tags como:

TITLE - Para adicionar um título ao qual irá aparecer na aba da página em um navegador.

META - Será o responsável pela definição de qualquer informação de metadados, como o atributo “charset”, que pode indicar o formato de codificação de caracteres usado no documento. Como por exemplo: meta charset="utf-8".

LINK - Será usado para informar o caminho de determinado documento que será usado para a construção da página, como um documento CSS que precisará dos atributos rel, type e href(devendo conter o caminho para o documento), como no seguinte exemplo:
link rel="stylesheet" type="text/css" href="css/style.css"
BODY
Há incontáveis tags a ser utilizadas no “corpo” de um site para maior liberdade para desenvolver sua página, veja a seguir:

B -Texto negrito.
BR - Quebra de linha.
STRONG - Texto importante.
I - Texto em itálico.
EM - Texto enfâse.
MARK - Texto marcado.
SMALL - Texto pequeno.
DEL - Texto apagado.
INS - Texto inserido.
SUB - Texto subscrito.
SUP - Texto superscrito.
ABBR - Define uma abreviação ou acrônimo.
ADDRESS - Para endereço.
BDO (dir="rtl"/"ltr") - Texto da direita para a esquerda.
BLOCKQUOTE (cite=”site da fonte”) - Texto de citação com fonte.
CITE - Texto de citação.
Q - Texto entre aspas.
NAV - Define o navegador do site.
TABLE - Gerar uma tabela.
TR - Gerar a linha de uma tabela.
TH - Gerar cabeçalho da tabela.
TD - Gerar célula de uma tabela.
UL - Gerar lista não-ordenada.
OL - Gerar lista ordenada. Ao utilizar o atriputo type, pode-se alterar o tipo de ordem da lista, como para letras (a) e números romanos (I). Como a seguir: ol type="a".
LI - Gerar um item para uma lista.
DL - Gerar lista detalhada.
DT - Gerar item da lista detalhada.
DD - Gerar detalhe do item da lista detalhada.
